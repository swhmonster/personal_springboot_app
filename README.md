# personal_springboot
>个人全栈开发实战：麻雀虽小，五脏俱全，精简全套项目开发，扫一扫程序员日常工作中的知识盲区

该应用已下线，网站以切流至新应用
项目网站地址：[~~https://waltersun.cn~~](https://waltersun.cn)
## 技术架构
- 总体架构
    - 前后端分离：Spring Boot + Vue-Cli + VuePress
    - 数据库：postgresql
    - 代理：nginx
    - 缓存：jetCache
    - 解耦、异步、削峰：rocketmq
## 涉及到的核心算法
>加解密模式支持ECB、CBC
- 国密算法SM2
- 国密算法SM3
- 国密算法SM4
